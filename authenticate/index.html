Authenticate Mopidy with music services.

Some services require you to get authentication tokens for Mopidy so that it can connect to your service accounts. This page helps you on the way.

Photo by Sergiu Bacioiu

SoundCloud

If you want to use the SoundCloud extension, you must first authenticate your Mopidy installation to access your SoundCloud account.

  1. No information is shared with the Mopidy web site.

  2. Add the token to the soundcloud/auth_token config value in your mopidy.conf file:

    [soundcloud]
auth_token = ...token goes here...

For more information see the Mopidy-SoundCloud docs.

Spotify

If you want to use the Mopidy-Spotify-Web extension, you must first authenticate your Mopidy installation to access your Spotify account.

  1. Your Spotify login details are never shared with this site.

  2. Add the credentials to the spotify_web config value in your mopidy.conf file:

    [spotify_web]
client_id = ...client_id goes here...
client_secret = ...client_secret goes here...

For more information see the Mopidy-Spotify-Web docs.
