Mopidy ♪

Mopidy is a music server which can play music from Spotify, or from your local hard drive.

Skip to latest release docs or development docs

It's a server

Mopidy is a server without any graphical user interface. It runs on the Linux computer with the speakers.

Mopidy keep track of the music, your playlists, and the clients that commands it.

Use your favorite client

You can use your favorite MPD client to search for music and manage the playlist. MPD clients are available for many platforms, including Windows, Mac OS X, Linux, Android, and iOS.

In other words, you can control the music from any computer or smartphone in your home.

Stream from Spotify

If you got a subscription for Spotify Premium, then Mopidy can stream music from Spotify's archive of 10 million tracks. You'll never need to refill your collection with new music again.

Install the latest release

Running Ubuntu?

If you're running Ubuntu you can install Mopidy from APT in a matter of seconds. Copy the following commands into the terminal, then follow the link in step 5.

  1. Add APT repository key wget -q -O - http://apt.mopidy.com/mopidy.gpg | sudo apt-key add -
  2. Add APT repository sudo wget -q -O /etc/apt/sources.list.d/mopidy.list http://apt.mopidy.com/mopidy.list
  3. Update package cache sudo apt-get update
  4. Install Mopidy sudo apt-get install mopidy
  5. Continue with the the settings
